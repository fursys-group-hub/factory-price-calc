### 치명 · Dockerfile(도커 설정) 만들기
아래 글을 그대로 복사해 AI 도구에 붙여넣으면 이 문제를 고쳐줍니다.
```
이 프로젝트는 index.html 한 장으로 된 정적 웹페이지야. 사내 서버(단일 컨테이너)에 올릴 수 있도록 레포 루트에 Dockerfile을 만들어줘. 요구사항:
- nginx 안정 버전(예: nginx:1.27-alpine) 기반으로 index.html 같은 정적 파일을 서비스
- 정적 파일을 /usr/share/nginx/html 로 복사
- EXPOSE 80
- 컨테이너가 nginx를 포그라운드로 실행: CMD ["nginx","-g","daemon off;"]
- 상태 점검(HEALTHCHECK)으로 / 경로가 200을 주는지 확인하는 줄도 추가
docker-compose나 빌드팩 말고 Dockerfile 한 개만 만들어줘.
```
